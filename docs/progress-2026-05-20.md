## 一、本次完成的工作

### 1. 浏览器渲染截图 Bug 修复（根因修复）

**问题**：Playwright `canvas.screenshot()` 截取的 canvas 内容全黑/全白，导致所有 CV 对比指标失效。

**根因链**：
1. WebGL 默认 `preserveDrawingBuffer: false`，帧缓冲区在渲染后立即清除
2. Playwright `canvas.screenshot()` 使用 OS 合成层捕获，与 WebGL buffer 不同步
3. `page.screenshot()` 截取整页（含 UI），非 canvas 内容
4. Three.js `renderer.setSize()` 在 `setPixelRatio(1)` 后需 `updateStyle=true`
5. 动画循环（requestAnimationFrame）持续覆盖手动设置的 `u_time`

**修复方案**：

| 文件 | 修复 |
|------|------|
| `shader-renderer.ts` | `preserveDrawingBuffer: true` + `resize()` 设 `pixelRatio=1` + `setTime()` 更新 `u_resolution` |
| `ShaderPreview.tsx` | `__setShaderTime` 先 `stopRendering()` 再渲染单帧 |
| `browser_render.py` | 改用 `canvas.toDataURL()` + `PIL.FLIP_TOP_BOTTOM` 垂直翻转 |

**验证**：1024×1024 渐变着色器 9/9 像素精确匹配（R/G/B 各通道误差 <5%）。

### 2. CV 对比重构：像素级 → 语义级

**动机**：参考帧（1280×720 16:9）与渲染帧（1024×1024 1:1）宽高比不同，像素级对比（MSE/SSIM/Edge IoU）无意义。

**新方案**：只保留宽高比无关的语义维度对比：

| 维度 | 指标 | 宽高比敏感？ |
|------|------|-------------|
| 色彩 | 分布相似度（归一化直方图余弦） + 主色重合度（Top-5 色距离） | ✅ 不敏感 |
| 亮度 | 分布相似度 + 均值偏差 + 高亮面积差 | ✅ 不敏感 |
| 纹理 | FFT 频谱归一化分布差异 + 复杂度匹配 | ✅ 不敏感 |
| 空间 | 主体中心距离（归一化坐标） + 面积占比差 + 边缘分布模式 | ✅ 不敏感 |

**移除的指标**：MSE、SSIM、Edge IoU、subject/background MSE、像素级区域对比。

## 二、V4 批量测试结果

**测试配置**：5 样本 / max_iter=3 / threshold=0.85

| Sample | Category | V2(无CV) | V3(旧CV) | **V4(新CV)** | V4-V2 |
|--------|----------|---------|---------|-------------|-------|
| electron | particle | 0.68 | 0.45 | **0.78** | **+0.10** |
| liquid-galss-test | liquid | 0.52 | 0.32 | **0.68** | **+0.16** |
| auroras | space | 0.42 | 0.58 | **0.62** | **+0.20** |
| cool-s-distance | warp | 0.52 | 0.42 | **0.42** | -0.10 |
| vortex-street | liquid | 0.81 | 0.78 | **0.24** | -0.57 |
| **Average** | | **0.59** | **0.51** | **0.55** | -0.04 |

**关键发现**：
- 截图 bug 修复后 CV 数据首次基于正确的渲染内容，可信度大幅提升
- **auroras、electron、liquid-galss-test 明显提升**（+0.10 ~ +0.20），语义级 CV 对比提供了有用的修正信号
- **vortex-street 异常下降**（0.81→0.24），超出 LLM 非确定性正常范围（±0.05），需单独分析
- **cool-s-distance 持平**，warp 类效果一直是难点

**报告路径**：
- V4 批量报告：`backend/test_e2e_results_v4/index.html`
- 单样本详细报告（auroras）：`backend/test_e2e_reports/auroras/report.html`
